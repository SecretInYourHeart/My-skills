# 软件产品技术参数表生成

> 根据《软件功能说明书》或《产品白皮书》，自动生成结构化技术参数表，直接用于招标文件或内部文档编制。

## 功能特性

- 📄 **多源文档支持** — 兼容 Word / PDF / 文本等多种格式的功能说明书、白皮书
- 📊 **三种输出格式** — 标准表格（可粘贴 Excel）、HTML 文件（含下载按钮）、纯文本
- 🔧 **自定义列结构** — 支持用户指定自定义列名、合并规则与取值规范
- 📋 **多产品对比** — 可将多个产品文档合并生成对比表，逐项标注满足情况
- ✅ **质量检查** — 输出前自动校验层级结构、取值规范、序号连续性等

## 快速使用

在 WorkBuddy 中直接输入以下任一触发词，并附上源文档：

| 触发词示例 |
|-----------|
| "帮我生成技术参数表" |
| "把这个功能说明书转成表格" |
| "根据白皮书整理功能清单" |
| "生成招标用的技术指标表" |
| "功能点整理" |

## 默认列结构

| 列名 | 说明 |
|------|------|
| 序号 | 全表递增编号 |
| 一级功能域 | 功能大分类（相同内容自动合并单元格） |
| 二级功能模块 | 功能子分类 |
| 功能点名称 | 具体功能点名称 |
| 技术参数与功能要求 | 详细技术描述与要求 |
| 是否必须 | 强制程度标识（★必须 / 必须 / 应当） |
| 备注 | 补充说明 |

## "是否必须"取值规范

| 取值 | 含义 |
|------|------|
| ★必须 | 核心功能，不可缺失 |
| 必须 | 重要功能，应包含 |
| 应当 | 建议功能，优先包含 |

## 输出格式说明

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| 可直接复制的标准表格 | Markdown 表格 | 快速预览、编辑 |
| 带下载按钮的 HTML 文件 | 独立 HTML，内嵌 CSS，含 Excel 导出按钮 | 正式文档、归档 |
| 纯表格文本 | 制表符分隔，无格式语法 | 特殊系统兼容 |
