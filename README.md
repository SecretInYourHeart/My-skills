# My-skills

【功能简介】
本Skill用于根据软件功能说明书/产品白皮书，生成可直接用于招标或内部文档的标准化技术参数表，支持规范排版、单元格合并、强制等级标注等专业格式输出。

【适用场景】
- 招投标文件编制
- 内部技术评审文档
- 产品立项材料
- 软件验收测试用例梳理

【核心特性】
1. 通用无指向：无项目/产品名称，适配所有软件系统
2. 标准列结构：序号/一级功能域/二级功能模块/功能点名称/技术参数与功能要求/是否必须/备注
3. 强制等级规范：统一使用★必须/必须/应当标识
4. 自动排版：一级功能域相同内容自动合并单元格
5. 多格式输出：支持纯文本粘贴版与标准Excel文件

【使用方式】
1. 上传软件功能说明书或产品白皮书
2. 选择输出样式（标准招标/内部文档/极简清单）
3. 一键生成技术参数表
4. 直接复制粘贴到Excel，或下载标准文件使用
